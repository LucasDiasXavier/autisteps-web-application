# 🧩 AutiSteps

## 📖 Sobre o Projeto

![AutiSteps](https://img.shields.io/badge/Projeto-Tecnologia%20Educacional-blue)
![Status](https://img.shields.io/badge/Status-Em%20Desenvolvimento-orange)

O **AutiSteps** é uma aplicação web desenvolvida para auxiliar crianças com Transtorno do Espectro Autista (TEA) na organização de rotinas diárias por meio de uma interface visual, intuitiva e acessível.

A plataforma foi projetada para tornar tarefas do cotidiano mais previsíveis e compreensíveis, promovendo autonomia, independência e desenvolvimento de habilidades essenciais para o dia a dia.

---

## 🎯 Objetivos

- Organizar atividades diárias de forma visual.
- Incentivar a autonomia da criança.
- Facilitar a compreensão da rotina.
- Reduzir a ansiedade causada por mudanças inesperadas.
- Promover o acompanhamento do progresso das tarefas.
- Utilizar recursos digitais para inclusão e desenvolvimento.

---

## ✨ Funcionalidades

### 📅 Rotina por Períodos

As atividades são organizadas em:

- ☀️ Manhã
- 🌤️ Tarde
- 🌙 Noite

Permitindo uma visualização clara e objetiva da rotina.

### 📊 Barra de Progresso

- Acompanhamento visual das tarefas concluídas.
- Atualização dinâmica do progresso.
- Incentivo à conclusão das atividades.

### 📱 Navegação Responsiva

- Menu adaptado para dispositivos móveis.
- Interface amigável e intuitiva.
- Experiência consistente em diferentes tamanhos de tela.

### 🎨 Interface Inclusiva

- Cores suaves e acolhedoras.
- Ícones intuitivos.
- Botões com boa área de interação.
- Layout pensado para facilitar a compreensão visual.

### ♿ Acessibilidade

- Estrutura semântica HTML5.
- Uso de atributos ARIA.
- Navegação por teclado.
- Elementos com foco visível.
- Contraste adequado para leitura.

## 🚀 Como Executar
O projeto não requer build ou pré-processadores. Siga os passos abaixo no seu terminal:

```bash
# Clone o repositório
git clone https://github.com/LucasDiasXavier/autisteps-web-application

# Entre na pasta do projeto
cd autisteps-web-application

# Abra o arquivo index.html no navegador

# Escolha o comando correspondente ao seu sistema:
open index.html    # macOS
start index.html   # Windows
xdg-open index.html # Linux

```
